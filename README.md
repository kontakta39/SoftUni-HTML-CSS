# 🎓 HTML & CSS – SoftUni Course Portfolio (January 2024)

## 📌 Overview

This repository contains my solutions and exercises completed during the [*HTML & CSS - January 2024 @ SoftUni*](https://softuni.bg/trainings/4361/html-and-css-january-2024) course.

The course focuses on building a solid understanding of modern web development fundamentals using **HTML5** and **CSS3**, with an emphasis on layout, styling, and responsive design principles.

---

## 🚀 Key Learning Outcomes

Throughout the course, I gained practical experience in:

* Structuring web pages using semantic HTML
* Applying CSS for styling, layout, and visual hierarchy
* Working with the CSS box model and positioning concepts
* Building flexible layouts using Flexbox
* Creating responsive designs using media queries and mobile-first principles
